## Features

- **Nostr NIP Compliance**: Full support for essential Nostr Improvement Proposals
- **Distributed Storage**: CockroachDB backend for data consistency and availability
- **Event Filtering**: Advanced filtering capabilities for efficient event retrieval
- **Real-time WebSocket**: High-performance WebSocket connections for live updates
- **HTTP API**: RESTful endpoints for relay management and monitoring
- **Metrics & Monitoring**: Prometheus metrics and structured logging
- **Docker Support**: Container-first deployment with orchestration support

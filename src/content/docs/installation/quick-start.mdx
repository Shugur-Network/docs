# Quick Start (Recommended)

The fastest way to deploy Shugur Relay.

## Standalone

```bash

```

## Distributed

```bash
curl -fsSL https://github.com/Shugur-Network/relay/raw/main/scripts/install.distributed.sh | sudo bash
```

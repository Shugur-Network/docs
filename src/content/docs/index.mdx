## Why Shugur Relay?

Shugur Relay is designed for operators who need reliability, observability, and horizontal scale in their Nostr infrastructure. Built for production use with enterprise-grade features.

<CardGrid>
  <Card title="Enterprise Grade" icon="star">
    Built with Go for performance and CockroachDB for distributed storage. Battle-tested in production with 35+ NIP support.
  </Card>
  <Card title="Advanced Features" icon="puzzle">
    Complete feature set including Time Capsules, Cashu Wallets, Nutzaps, Lightning Zaps, and Moderated Communities.
  </Card>
  <Card title="High Performance" icon="rocket">
    50,000+ concurrent connections, 25,000+ events/sec in distributed mode. Optimized for massive scale.
  </Card>
</CardGrid>
